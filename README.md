# Task Scheduler (任务调度器)

基于 Electron + Node.js 的本地自动化任务管理工具。支持创建定时任务、管理永久任务，并提供日历视图进行任务概览。

## 核心功能

### 1. 任务类型支持
*   **HTTP 请求**: 定时发送 GET/POST/PUT/DELETE 请求。支持自定义 Headers、Cookies 和 Body (JSON)。
*   **运行程序 (EXE)**: 定时启动本地应用程序，支持传递启动参数。
*   **系统弹窗**: 定时在桌面弹出提醒消息。
*   **日志/备忘**: 在日历上记录备忘信息（仅作为记录，不执行）。

### 2. 任务调度模式
*   **永久运行**: 任务会无限期按照设定的间隔（Cron 表达式）循环执行。
*   **日期范围运行**: 设定开始日期和结束日期，任务仅在该时间范围内执行。
*   **自定义间隔**: 支持 秒/分/时/天 等维度的执行频率设置。

### 3. 界面特性
*   **三栏布局**:
    *   **左侧**: 永久任务列表 & 创建任务入口。
    *   **中间**: 月视图日历，直观展示每天的任务安排。
    *   **右侧**: 选中日期的任务详情列表。
*   **迷你模式 (Mini Mode)**:
    *   一键切换至精简窗口 (240x500)。
    *   窗口置顶显示。
    *   仅显示今日任务，支持快速删除。
    *   自动退出全屏，避免遮挡。
*   **日历概览**: 每天的任务会以彩色小条目显示在日历格子中，支持点击查看详情。

## 开发与运行

### 环境要求
*   Node.js (建议 v16+)
*   npm

### 安装依赖
```bash
npm install
```

### 启动开发环境
```bash
npm run start
```

### 打包 (Windows)
生成便携版 EXE 文件（无需安装）：
```bash
npm run dist
```
打包后的文件位于 `dist-release` 目录。

## 技术栈
*   **Electron**: 桌面应用框架
*   **TypeScript**: 强类型语言支持
*   **Node-Schedule**: 定时任务调度核心
*   **Node-Notifier**: 系统通知支持
*   **Electron-Builder**: 应用打包

## 注意事项
*   任务数据默认保存在项目根目录的 `tasks.json` 文件中。
*   迷你模式下隐藏了编辑和启动/停止功能，仅供查看和快速删除。
